# X Mandamentos (falíveis) para os tempos modernos:

Apenas algumas sugestões (empíricas) para usufluir o melhor da Internet.

- 1. Não ofenda o próximo. -> Somos todos irmãos
- 2. Sempre utilize sua cognição e lógica -> não acredite em tudo.
- 3. Fazeis back-up do que é importante -> Evite ser surpreendido.
- 4. Nunca utilizar a mesma senha em dois lugares -> Evite ser surpreendido (x2). 
- 5. Sempre utilize todos os métodos de segurança disponíveis -> Evite ser surpreendido (x3).
- 6. Gaste pouco (ou não gaste) tempo nas redes sociais -> Leia um livro, converse com seus amigos. 
- 7. Cuide de seu dinheiro e de seus dados -> Data is the new Oil.
- 8. Sempre prefira o open source -> Tudo que é aberto/auditavel é mais confiavel.
- 9. Sempre leia os Termos dos serviços (TOS) -> Evite ser surpreendido (x4).
- 10. Você nunca esta totalmente seguro -> Sempre se aprimore, é um círculo contínuo.
